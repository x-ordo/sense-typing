# PROJECT CONTEXT: Sense Typing

## 1. Vision
**"Typographic Intelligence."**
단순한 폰트 리스트를 넘어, AI가 사용자의 디자인 의도를 분석하여 최적의 타이포그래피 정체성을 결정해주는 차세대 폰트 커머스 플랫폼.

## 2. Core Identity
- **Luxury Editorial UI:** 고대비 테마와 비대칭 그리드를 활용한 전문 디자인 에이전시 룩앤필.
- **Intelligence-First:** 모든 검색과 추천은 AI(Gemini) 분석을 기반으로 함.
- **Direct Commerce:** 복잡한 설명 없이 상품(폰트)과 라이선스 정보에 즉시 접근.

## 3. Tech Stack
- **Frontend:** Next.js 15 (App Router), Tailwind CSS
- **Runtime:** Cloudflare Edge Runtime
- **AI:** Google Gemini 1.5 Flash
- **Icons:** Lucide React
- **Deployment:** Cloudflare Pages
