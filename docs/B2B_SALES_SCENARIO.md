# B2B Sales Scenario: Sense Typing for Enterprise

## 1. Value Proposition (핵심 가치)
"조직의 타이포그래피 리스크를 0으로 줄이고, 브랜드 일관성을 AI로 통제하십시오."

## 2. Target Segments (타겟 세그먼트)
- **에이전시/스튜디오:** 수많은 클라이언트 프로젝트의 폰트 라이선스 관리를 단일 플랫폼에서 해결.
- **스타트업/중소기업:** 전문 디자이너 없이도 AI를 통해 수준 높은 브랜드 아이덴티티 구축.
- **대기업 법무/홍보팀:** 전사적으로 사용되는 폰트의 상업적 이용 가능 여부 모니터링 및 표준화.

## 3. Pain Points & Solutions (문제 해결)

| Pain Point (고충) | Sense Typing Solution (해결책) |
| :--- | :--- |
| 어떤 폰트가 상업적으로 안전한지 찾기 힘듦 | **License Matrix**와 **Admin Review**를 통과한 검증된 데이터만 제공 |
| 팀원마다 사용하는 서체가 달라 브랜드가 파편화됨 | **Admin Pipeline**을 통한 조직 전용 '승인된 폰트' 리스트 운영 |
| 폰트 구매 비용 및 옵션 계산이 복잡함 | **Market Simulator**를 통해 용도별 비용 즉시 산출 및 라이선스 구매 |

## 4. Revenue Model (수익 모델)
1. **Premium License Fee:** 개인/기업별 유료 폰트 단건 판매 수수료.
2. **Subscription (SaaS):** 기업용 중앙 관리 대시보드 및 AI 고급 분석 기능 구독.
3. **Featured Placement:** 신규 폰트 제작사 대상의 상단 노출 광고 모델.

## 5. Success Metric (성공 지표)
- 사용자가 폰트 선택 후 상세 페이지(구매/다운로드)로 전환되는 비율 (Conversion Rate).
- AI 추천 결과에 대한 사용자 만족도 및 재방문율.
- 어드민 파이프라인을 통한 데이터 업데이트 속도 및 무결성.
