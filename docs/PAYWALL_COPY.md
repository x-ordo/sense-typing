# 🔐 Paywall Copywriting Strategy: The "Risk-Free" Decision

## 1. Core Positioning
**이 서비스는 폰트를 파는 것이 아니다. '결정 실패 리스크'를 제거해주는 보험이다.**

> **User Mindset:** "폰트가 예뻐서 산다" (X) → "잘못 골라서 욕먹거나 문제가 생기는 게 무섭다" (O)

---

## 2. Copy Sets (A/B Testing Candidates)

### Version A: The "Responsibility Transfer" (책임 전가형)
*Target: 실무자, 중간 관리자 (욕먹기 싫은 사람)*

**Headline:**
> **"폰트 선택이 아니라, 책임 이전입니다."**

**Sub-headline:**
> 정부 문서, IR, 계약서, 투자자료...
> 이 엔진은 **‘문제가 생겼을 때 설명 가능한 선택’만** 제공합니다.
> 당신의 감이 아니라, 검증된 데이터 뒤에 숨으세요.

**CTA:**
> [ 안전한 결정 권한 구매하기 (월 9,900원) ]
> *커피 2잔 값으로 반려될 확률을 지우세요.*

---

### Version B: The "Cost of Failure" (비용 회피형)
*Target: 창업자, PM, 리더 (결과가 중요한 사람)*

**Headline:**
> **"틀리면 비용이 나는 문서에 쓰는 엔진입니다."**

**Sub-headline:**
> 무료 폰트 뒤지다가 3시간 날리셨나요?
> 아니면 잘못된 폰트로 신뢰를 잃으셨나요?
> **비즈니스 문맥(Context)에 맞는 폰트는 따로 있습니다.**

**CTA:**
> [ 실패 비용 제거하기 (월 9,900원) ]
> *잘못된 폰트로 잃을 수 있는 기회비용보다 쌉니다.*

---

## 3. Feature-to-Value Translation (기능의 가치화)

| Feature (기능) | Value Proposition (가치 제안) | Copy Example |
| :--- | :--- | :--- |
| **Emotion Tag Filtering** | 감성 제거, 목적 집중 | "예쁜지 고민하지 마세요. **통하는지**만 봅니다." |
| **Use Case Mapping** | 상황별 안전장치 | "IR 덱에 귀여운 폰트를 쓰는 실수를 원천 차단합니다." |
| **High Confidence Score** | 검증된 다수의 선택 | "이미 1,000건의 성공한 문서가 선택한 폰트입니다." |
| **Tag/Reason Locking** | 논리적 근거 제공 | "상사가 '왜 이거 썼어?' 물을 때 대답할 근거를 드립니다." |

---

## 4. Trigger Points (Paywall 노출 시점)

1.  **Use Case 필터 클릭 시 (결정적인 순간)**
    *   *Copy:* "이 상황(정부 문서/IR)에 검증된 리스트를 보시겠습니까?"
2.  **검색 결과 2위 이하 열람 시 (비교의 순간)**
    *   *Copy:* "상위 1개는 무료입니다. 더 정교한 대안이 필요하신가요?"
3.  **'이 폰트를 선택한 이유' 클릭 시 (확신의 순간)**
    *   *Copy:* "이 폰트가 왜 안전한지, **데이터로 증명된 이유**를 확인하세요."

---

## 5. Pricing Structure (Risk-Based Pricing)

### **Personal Plan (Risk Avoidance)**
*   **Price:** ₩9,900 / 월
*   **Key Value:** "한 번의 잘못된 선택 비용보다 쌉니다."
*   **Includes:**
    *   모든 Use Case 필터 해제
    *   Risk Level 3 이하 접근 가능
    *   폰트 선택 근거 리포트 (요약)

### **Team Plan (Decision Standardization)**
*   **Price:** ₩49,000 / 월 (5인 기준)
*   **Key Value:** "우리 팀 문서의 톤앤매너, 이 기준으로 통일합니다."
*   **Includes:**
    *   Risk Level 5 (최고 위험군) 접근 가능
    *   팀원 초대 및 라이선스 공유
    *   산업별 추천 프리셋 제공

---

## 6. Closing Statement (Internal Note)

> 고객은 **결과**를 산다.
> 폰트는 **수단**일 뿐이다.
> 우리는 **결과의 확실성**을 판다.
