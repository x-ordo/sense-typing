## 0. Executive Summary

**이 프로젝트의 상품은 폰트가 아니다.**
상품은 `Taxonomy(분류 체계)`이며, 수익원은 `결정 권한`이다.

* Noonnu: 폰트 **저장소(Data)**
* 본 프로젝트: 폰트 **결정 엔진(Insight)**

> 사용자는 폰트를 고르지 않는다. **위험을 회피하고, 결과를 산다.**

## 5. DB Schema (Supabase)

```sql
create table fonts (
  id bigint generated by default as identity primary key,
  name text not null,
  developer text,
  license_summary jsonb,
  preview_url text,
  download_link text,
  created_at timestamp with time zone default timezone('utc', now())
);

create table tags (
  id bigint generated by default as identity primary key,
  name text not null,
  category text not null,
  intent text,
  is_premium boolean default false
);

create table font_tags (
  font_id bigint references fonts(id),
  tag_id bigint references tags(id),
  score int default 3,
  primary key (font_id, tag_id)
);

create table users (
  id uuid primary key,
  email text,
  plan text default 'free'
);
```
