## 라이선스 유형

- **OFL (Open Font License)**: 자유롭게 사용, 수정, 배포 가능
- **Free**: 무료 사용 가능, 제작사 조건 확인 필요
- **Commercial Free**: 상업적 무료 사용 가능

## 프로젝트 반영 방법

1. **mock-data.ts 업데이트**: 위 TypeScript 데이터를 `src/lib/mock-data.ts`에 추가
2. **Supabase seed 업데이트**: `supabase/seed.sql`에 INSERT 문 추가
3. **데이터베이스 마이그레이션**: 새로운 폰트 데이터를 production DB에 추가
